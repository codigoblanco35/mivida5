Expectativas de Aprendizaje - Diseño y Desarrollo Web

Estoy emocionado por aprender nuevas técnicas de diseño y desarrollo web en este curso. Espero adquirir habilidades prácticas que me permitan crear sitios web atractivos y funcionales.

Planeo utilizar lo aprendido en clase para mejorar la usabilidad y el diseño de mis proyectos web. Desde la selección de colores hasta la disposición de elementos en la página, espero aplicar principios de diseño efectivos para crear experiencias web memorables.

Además, quiero comprender mejor los aspectos técnicos del desarrollo web, como HTML, CSS y JavaScript, para poder personalizar mis proyectos y hacerlos más interactivos y dinámicos.

En resumen, espero que este curso me proporcione las herramientas y el conocimiento necesarios para convertirme en un diseñador y desarrollador web competente y creativo.
